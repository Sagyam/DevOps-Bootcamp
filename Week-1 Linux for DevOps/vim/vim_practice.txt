 VIM PRACTICE  --  "Search with your fingers, not your eyes"
==================================================================
RULES OF THE GAME
  * Press Esc to make sure you are in NORMAL mode before each drill.
  * NO arrow keys. NO mouse. NO scroll wheel. Hands stay on home row.
  * If you get lost, gg goes to the top, G goes to the bottom.
  * Bail out any time with  :q!   (quit, discard everything).

Work top to bottom. Each drill tells you the keys and the target.
------------------------------------------------------------------

DRILL 1 - TOP & BOTTOM
  Press  G  to fly to the last line, then  gg  to fly back here.

DRILL 2 - SEARCH + REPEAT  ( / n N )
  Type  /ERROR  then Enter. Press  n  to walk forward through every
  ERROR below, and  N  to walk back. There are five of them.
  ERROR one ......... ERROR two ......... ERROR three
  ......... ERROR four ......... ERROR five .........

DRILL 3 - THE NEEDLE  ( / )
  Somewhere below is the word NEEDLE exactly once. Find it with
  /NEEDLE  in a single keystroke-burst instead of scanning by eye.

DRILL 4 - WORD UNDER CURSOR  ( * and # )
  Park the cursor on the word config here: config
  Press  *  to jump to the NEXT config, and  #  to jump back.
  config appears again here, and once more here: config.

DRILL 5 - FIND CHAR ON A LINE  ( f ; , )
  GET,POST,PUT,PATCH,DELETE,HEAD,OPTIONS,CONNECT,TRACE,QUERY
  Put the cursor at line start (0). Press  f,  to hop to the first
  comma, then  ;  ;  ;  to hop comma-to-comma. Press  ,  to reverse.

DRILL 6 - TILL CHAR  ( t )
  /usr/local/bin/orbit/scripts/deploy.sh
  Press  t/  to stop JUST BEFORE the next slash. Then  ;  to repeat.

DRILL 7 - WORD MOTIONS  ( w b e )
  The quick brown fox jumps over the lazy sleeping orbit daemon.
  Hop with  w  (next word),  b  (back),  e  (end of word).

DRILL 8 - LINE ENDS  ( 0 ^ $ )
        this line is indented with spaces before the text ends here.
  Press  0  (column zero),  ^  (first non-blank),  $  (line end).

DRILL 9 - MATCHING BRACKETS  ( % )
  data = transform( parse(input)[0], { mode: "fast", retries: 3 } )
  Put the cursor on any  ( [ {  and press  %  to leap to its partner.

DRILL 10 - JUMP TO A LINE NUMBER  ( :NN )
  Run  :63  to land exactly on the TARGET line below. (It IS line 63.)
